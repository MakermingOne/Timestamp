# 时光印记（网站框架）

当前前端版本已实现：
- 上传图片并展示全部缩略图（Free 最多 50 张，Pro 最多 200 张）
- 点击缩略图预览并支持上一张/下一张切换
- 页面底部明显主按钮：**开始添加时间戳**
- 默认右下角无底色水印（仿胶片日期风格字体）
- 基础设置细分为：时间戳格式、水印位置、时间批量修改
- 支持单张照片独立修改日期/时间，并提供“确认应用”“恢复默认”
- 高级功能列表带 Pro 标识；未订阅时置灰，点击会打开订阅弹窗
- 使用底部提示气泡替代突兀的“指令输出/结果输出”框

## 本地运行

```bash
python -m http.server 4173
```

访问 `http://localhost:4173`
