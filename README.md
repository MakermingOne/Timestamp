# 时光印记·Timestamp

> 时间水印批量添加，成长每一步都有迹可循

一款轻量化、无门槛的网页端工具，用于批量读取照片 EXIF 拍摄时间并自动添加时间戳/成长水印。

![Version](https://img.shields.io/badge/version-v1.9-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 功能特性

### Free 版
- 📤 批量上传（最多 20 张）
- 📷 自动读取 EXIF 拍摄时间
- 🌍 支持多种国际时间格式（ISO、美式、英式、中文等）
- 🎯 可视化九宫格位置选择器
- 📏 边缘距离自定义（默认 3mm）
- 📋 EXIF 信息展示（拍摄时间、相机型号、分辨率、文件大小）
- 🖼️ 固定尺寸预览（600×400）
- 💾 导出功能（当前/所有≤20张/打包 ZIP）
- 🎞️ 胶片风格时间戳字体

### Pro 版
- 📤 批量上传提升至 **200 张**
- 👶 宝宝成长水印（支持男宝/女宝/中性头像 Logo）
- 📝 成长印记换行显示（印记在上，时间在下）
- ⏰ 单张/批量时间修改
- ⏱️ 时间批量偏移（加减天数/小时）
- 💾 批量压缩导出（可设置最大输出体积）
- 🎨 自定义时间格式
- 🖼️ 宝宝头像 Logo（可选显示/不显示）

## 🚀 快速开始

### 本地运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 4173

# 或使用 Node.js
npx serve -s -l 4173
```

访问 `http://localhost:4173`

### 在线使用
直接打开 `index.html` 文件即可使用（部分功能需要本地服务器支持）

## 📖 使用指南

### 1. 上传照片
- 拖拽或点击上传区域选择照片
- 支持 JPG、JPEG、PNG 格式
- 预览区显示 600×400 固定尺寸缩略图
- Free 版最多 20 张，Pro 版最多 200 张

### 2. 配置水印
- 选择时间格式（支持自定义格式）
- 调整水印位置（九宫格或拖拽）
- 设置边缘距离（默认 3mm）
- Pro 用户可配置宝宝成长水印和头像 Logo

### 3. 导出照片

| 功能 | 说明 | Free 限制 |
|------|------|-----------|
| 导出当前 | 导出当前预览的照片 | 无限制 |
| 导出所有 | 导出所有照片到文件夹 | ≤20张 |
| 打包导出 | 打包为 ZIP 下载 | ≤20张 |

**提示**：Free 用户导出超过 20 张时，可逐个导出当前照片，或升级 Pro。

## 📝 技术栈

- 纯前端实现，无需后端
- 使用 `exifr` 库读取 EXIF 信息
- 使用 `jszip` 库打包导出
- 原生 JavaScript + CSS3

## 🔒 隐私说明

- 所有处理均在浏览器本地完成
- 照片不会上传到任何服务器
- EXIF 信息仅用于显示和时间戳生成
