# 时光印记·Timestamp v1.9 🎉

> 时间水印批量添加，成长每一步都有迹可循

一款轻量化、无门槛的网页端工具，用于批量读取照片 EXIF 拍摄时间并自动添加时间戳/成长水印。

![Version](https://img.shields.io/badge/version-v1.9-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## ✨ 核心功能

### 📸 照片上传与处理
- 支持批量上传（Free: 20张 / Pro: 200张）
- 自动读取 EXIF 拍摄时间作为默认时间戳
- EXIF 缺失时智能回退到文件修改时间
- 实时缩略图预览与选择

### 🎨 多样化时间格式
支持 8 组国际通用格式：

| 格式组 | 示例 |
|--------|------|
| 点号分隔（默认）| `YYYY.MM.DD HH:mm` |
| 国际标准 | `YYYY-MM-DD HH:mm` |
| 斜杠分隔 | `YYYY/MM/DD HH:mm` |
| 美式格式 | `MM/DD/YYYY hh:mm AM/PM` |
| 英式/欧式 | `DD/MM/YYYY HH:mm` |
| 中文格式 | `YYYY年MM月DD日` |
| 紧凑格式 | `YYYYMMDD` |
| **自定义** | 支持 `YYYY/MM/DD/HH/mm/ss/AM/PM` 占位符 |

### 🎯 可视化位置控制
- **九宫格位置选择器**：左上/中上/右上/左中/居中/右中/左下/下中/右下
- **边缘距离自定义**：支持 mm/px 双单位，**默认 3mm**
- **拖拽微调**：预览区直接拖动水印位置

### 🎞️ 胶片风格水印
- 使用经典等宽字体（Courier New）
- 模拟旧式胶片相机时间戳效果
- 14px 精致字号，白色高对比度显示

## 👶 Pro 高级功能（v1.9 新版）

### 🍼 宝宝成长水印
- **自动年龄计算**：拍摄时宝宝年龄（X岁X个月X天）
- **头像 Logo**：可选男宝/女宝/中性头像，或不显示
- **显示模式**：时间戳 + 成长印记，或单独显示
- **格式模板**：支持昵称前缀、多种时间表达方式

### 💾 批量压缩导出
- **体积限制**：可设置单张最大输出体积（500KB ~ 10MB）
- **智能压缩**：超过限制自动调整质量
- **批量处理**：一键处理所有照片

### ⏰ 时间批量修改
- **天数偏移**：批量加减天数
- **小时偏移**：批量加减小时
- **快速恢复**：一键恢复默认时间

```bash

cd Timestamp
```

## 🛠️ 技术栈

- 纯前端实现，无需后端
- 使用 `exifr` 库读取 EXIF 信息
- 使用 `jszip` 库打包导出
- 原生 JavaScript + CSS3

## 🔒 隐私说明

- 所有处理均在浏览器本地完成
- 照片不会上传到任何服务器
- EXIF 信息仅用于显示和时间戳生成
