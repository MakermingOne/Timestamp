# 时光印记（照片批量时间戳添加工具）产品需求文档 PRD

## 1. 产品概述

### 1.1 产品名称
**时光印记** - 照片批量时间戳添加工具

### 1.2 产品定位
时光印记是一款轻量化、无门槛的网页端工具，用于批量读取照片 EXIF 拍摄时间并自动添加时间戳/成长水印，覆盖个人照片整理、家庭相册冲印、生活记录归档等高频场景。

### 1.3 目标用户
- 个人用户：旅行、日常记录整理
- 家庭用户：相册冲印前统一时间信息
- 亲子用户：宝宝成长时间线记录与专属年龄水印

### 1.4 核心价值
- **高效**：支持批量上传与一键处理，无需逐张编辑
- **精准**：优先读取 EXIF 原始拍摄时间，降低人工输入误差
- **灵活**：支持时间格式、位置、单位（mm/px）和模板配置
- **便捷**：纯网页端，无需安装，适配多设备访问

## 2. 版本范围与里程碑

### 2.1 MVP（V1）
- 批量上传（最多 200 张）
- 自动读取 EXIF 拍摄时间并生成时间戳
- 支持基础水印设置（位置、格式、字体大小、颜色）
- 预览区可拖拽调整水印位置
- 分批处理与合并导出（ZIP）
- 进度条与处理状态反馈

### 2.2 增强版（V1.5）
- 宝宝成长水印（出生日期 + 昵称 + 模板）
- 自动计算拍摄时宝宝年龄（X岁X月 / X个月 / X天）
- 横竖图智能边距策略（离边缘约 2mm）
- 实时处理进度显示（如“已处理 120/200 张”）
- 处理完成后支持“分批次导出”与“合并导出”双模式

### 2.3 高级版（V2）
- 批次任务历史记录
- 自定义模板市场（时间样式/亲子贴纸）
- 云端存储与跨端同步（可选）

## 3. 核心功能需求

## 3.1 照片上传模块
### 功能描述
用户可一次性上传最多 200 张照片，系统生成缩略图并支持分页浏览。

### 详细需求
1. 支持拖拽上传与点击上传（JPG/JPEG/PNG，优先 JPG/JPEG EXIF）
2. 单次最多 200 张，超限需拦截并提示
3. 显示上传总数、已解析数、异常数
4. 缩略图按分页展示（建议每页 20 张）
5. 支持删除单张 / 清空全部 / 重传
6. 对 EXIF 缺失照片给出标识并启用回退策略

### EXIF 回退策略
- 优先：DateTimeOriginal
- 其次：CreateDate / ModifyDate
- 再次：文件创建时间（本地可读时）
- 最后：提示用户手动输入统一时间或跳过

## 3.2 预览与水印定位模块
### 功能描述
用户可在预览区查看效果，并通过拖拽或输入精确位置调整水印。

### 详细需求
1. 左侧预览区展示当前选中图片
2. 水印可拖拽（实时更新 x/y 坐标）
3. 支持位置输入框（上/下/左/右边距）
4. 支持单位切换：mm 与 px
5. 支持九宫格快捷定位（左上/中上/右上...右下）
6. 横图、竖图自动建议位置：默认右下，边距 2mm

## 3.3 基础设置模块
### 配置项
1. 时间格式：
   - YYYY.MM.DD
   - YYYY-MM-DD HH:mm
   - YYYY年MM月DD日
   - 自定义格式（限简化格式串）
2. 字体与样式：
   - 字号（8~72）
   - 字色（含透明度）
   - 阴影/描边开关
3. 水印位置：
   - 九宫格 + 精确输入
4. 单位：mm/px
5. 时间来源与编辑：
   - 使用 EXIF 原始拍摄时间（默认）
   - 使用用户修正时间（覆盖 EXIF）
   - 使用手动输入时间（无 EXIF 或用户指定）

### 时间编辑能力（新增）
- 支持单张照片修改“拍摄时间/水印时间”，修改后实时刷新预览
- 支持批量统一时间偏移（如：全部照片 +1 天 / -2 小时）
- 支持批量指定固定时间（谨慎操作，需二次确认）
- 支持“恢复为 EXIF 原始时间”一键回退
- 所有时间编辑结果需在缩略图列表中标记“已修改时间”状态

### 交互要求
- 任一设置修改后，预览区实时刷新
- 时间编辑后需显示“原始时间 → 当前水印时间”对照
- 提供“恢复默认”按钮

## 3.4 宝宝成长水印模块
### 功能描述
针对亲子场景，基于宝宝出生日期与照片拍摄日期计算成长年龄，并生成专属水印。

### 配置项
1. 出生日期（必填）
2. 宝宝昵称（可选）
3. 年龄显示规则（系统自动判断）：
   - 月龄模式（0~24个月）：`X 个月 X 天`（如：6 个月 12 天）
   - 年龄模式（>24个月）：`X 岁 X 个月`（如：2 岁 3 个月）
4. 模板选择：3 种宝宝专属模板（卡通字体 + 年龄 + 小图标）
   - 模板 A：奶瓶图标
   - 模板 B：星星图标
   - 模板 C：脚印图标
5. 叠加内容选择：
   - 仅年龄水印
   - 年龄 + 拍摄日期（示例：`2026-03-08 | 宝宝 6 个月 12 天`）
6. 样式配置：字体颜色、图标样式、水印位置（与基础时间戳共用定位能力）

### 年龄计算规则
- 以“拍摄日期 - 出生日期”计算
- 0~24个月：显示“X个月X天”
- >24个月：显示“X岁X个月”
- 跨月按自然月计算，天数按自然日补差
- 若拍摄日期早于出生日期，提示：`照片时间早于宝宝出生，无法计算年龄`
- 若无 EXIF 拍摄时间，支持用户手动输入照片拍摄日期后再计算年龄

## 3.5 时间修正与批量校时模块
### 功能描述
支持用户对 EXIF 拍摄时间进行修正，并将修正结果用于最终水印时间渲染，满足相机时间错误、时区错误、历史照片补录等场景。

### 详细需求
1. 时间编辑粒度：单张编辑、批量编辑（多选）
2. 时间编辑方式：直接选择日期时间 / 增减偏移量（天、小时、分钟）
3. 批量偏移示例：`+8 小时`（时区修正）、`+1 天`（相机日期错误）
4. 生效范围选择：仅当前选中 / 全部照片 / 当前页
5. 编辑历史：支持撤销上一步与清空本次编辑
6. 数据标记：记录 `originalExifTime` 与 `effectiveWatermarkTime`
7. 导出规则：默认仅改水印时间，不改原图 EXIF；若支持写回 EXIF，需单独开关并提示风险

## 3.6 批处理与导出模块
### 功能描述
提供“分批处理”与“合并导出”能力，确保大批量场景稳定。

### 详细需求
1. 分批处理按钮：默认每批 20 张（可配置）
2. 处理进度：实时显示“已处理 X/Y 张”（示例：`已处理 120/200 张`）并展示失败数
3. 失败重试：支持仅重试失败项
4. 导出方式：
   - 单批次 ZIP 下载（处理完成后可逐批下载）
   - 合并导出 ZIP（保留原文件名 + 后缀 `_timestamp`）
5. 处理完成提示：成功/失败统计

### 体验优化要求
- 处理过程中在底部固定显示全局进度条与当前进度文案
- 当前批次完成后可立即导出该批次，不必等待全部任务结束
- 全部任务完成后提供“合并导出”主入口，并提示总成功数/失败数

## 4. 前端页面与交互设计需求

## 4.1 视觉风格
- 整体风格：现代简洁、亲子/生活化
- 主色调：柔和蓝 + 暖白
- 宝宝模块强调色：浅粉
- 组件形态：卡片式、圆角、轻微卡通元素（仅宝宝模块）

## 4.2 页面布局
1. **顶部导航**：产品名、帮助入口、语言切换（可选）
2. **主体区域**：
   - 左侧：
     - 照片上传区
     - 缩略图分页区
     - 预览区（可拖拽水印）
   - 右侧：
     - 基础设置面板
     - 宝宝成长水印面板
3. **底部操作区**：
   - 主按钮：“分批处理”
   - 次按钮：“合并导出”
   - 全局进度条

## 4.3 响应式适配
- PC：双栏布局（预览与设置并行）
- 平板：上下布局（预览在上，配置在下）
- 手机：分步骤流程（上传 → 设置 → 预览 → 导出）

## 4.4 水印位置精准自定义
1. 单位能力：支持毫米(mm)/像素(px)双单位，默认毫米
2. 可视化调整：可在预览图直接拖动水印
3. 数值化调整：支持输入边距值（例：右 5mm、下 3mm）
4. 预设位置：右下角（默认）、左下角、正中央、右上角、左上角
5. 横/竖图适配：自动识别照片比例，切换图片时保持相对位置比例

---

## 5. 业务规则与异常处理

1. 上传超过 200 张：阻止并提示上限
2. 图片格式不支持：标记并跳过
3. EXIF 缺失：按回退策略处理并在 UI 标识
4. 用户修改拍摄时间后，系统以“当前水印时间”作为渲染依据
5. 若用户输入时间格式非法（如 2026-13-40），需阻止保存并提示修正
6. 水印超出边界：自动吸附到可视区域
7. 批处理异常：支持失败重试，不中断整体任务
8. 导出失败：给出错误原因与重试入口

---

## 6. 非功能需求

### 6.1 性能
- 200 张图片上传后首屏缩略图渲染 ≤ 3 秒（建议值）
- 单张加水印处理时延 ≤ 300ms（本地环境目标）
- 批处理支持断点续跑（前端任务队列）

### 6.2 可用性
- 核心流程（上传→设置→导出）在 3 分钟内可完成
- 新用户首次使用无需教程即可上手

### 6.3 兼容性
- 浏览器：Chrome / Edge / Safari 主流近两年版本
- 终端：桌面端优先，兼容平板与手机

### 6.4 隐私与安全
- 默认前端本地处理，不上传服务器（若技术方案允许）
- 如需云处理，需明确告知并提供隐私说明

## 7. 埋点与数据指标

### 7.1 关键埋点
- 上传按钮点击
- 上传成功张数
- EXIF 解析成功率
- 分批处理开始/完成/失败
- 导出点击与导出成功
- 宝宝水印模块使用率

### 7.2 核心指标（北极星 + 业务指标）
- 北极星：成功导出任务数 / 日
- 转化：上传后完成导出率
- 质量：处理失败率、EXIF 解析失败率
- 体验：平均处理时长、平均每次处理张数

---

## 8. 验收标准（UAT）

1. 可一次上传 200 张照片并稳定显示缩略图
2. 至少 90% 含 EXIF 的照片可正确读取拍摄时间
3. 水印位置可拖拽、可输入、可单位切换且一致生效
4. 宝宝年龄水印计算正确（覆盖跨年、跨月样例）
5. 时间编辑能力可用（单张修改、批量偏移、恢复 EXIF）且结果正确
6. 分批处理进度准确，失败项可重试
7. 导出 ZIP 可下载且文件名规则正确
8. PC/平板/手机均可完整走通核心流程
