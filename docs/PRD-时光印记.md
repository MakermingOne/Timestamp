# 时光印记（照片批量时间戳添加工具）产品需求文档 PRD

> 版本: v1.5 | 最后更新: 2026-03-12

---

## 1. 产品概述

### 1.1 产品名称
**时光印记** - 照片批量时间戳添加工具

### 1.2 产品定位
时光印记是一款轻量化、无门槛的网页端工具，用于批量读取照片 EXIF 拍摄时间并自动添加时间戳/成长水印，覆盖个人照片整理、家庭相册冲印、生活记录归档等高频场景。

### 1.3 目标用户
- 个人用户：旅行、日常记录整理
- 家庭用户：相册冲印前统一时间信息
- 亲子用户：宝宝成长时间线记录与专属年龄水印

### 1.4 核心价值
- **高效**：支持批量上传与一键处理，无需逐张编辑
- **精准**：优先读取 EXIF 原始拍摄时间，降低人工输入误差
- **灵活**：支持时间格式、位置、单位（mm/px）和模板配置
- **便捷**：纯网页端，无需安装，适配多设备访问

## 2. 版本范围与里程碑

### 2.1 Free 版（当前）
- 批量上传（最多 **20 张**）
- 自动读取 EXIF 拍摄时间并生成时间戳
- 支持多种国际时间格式（ISO、美式、英式、中文等）
- 可视化九宫格位置选择器
- 支持边缘距离自定义（mm/px）

### 2.2 Pro 版（V1.5）

- 自动计算拍摄时宝宝年龄（X岁X月 / X个月 / X天）
- 时间戳与成长水印合并显示
- 自定义格式支持（YYYY/MM/DD/HH/mm/ss/AM/PM 占位符）

### 2.3 高级版（V2 - 规划中）
- 批次任务历史记录
- 自定义模板市场（时间样式/亲子贴纸）
- 云端存储与跨端同步（可选）

## 3. 核心功能需求

## 3.1 照片上传模块
### 功能描述
用户可一次性上传照片，系统生成缩略图并支持分页浏览。

### 详细需求
1. 支持拖拽上传与点击上传（JPG/JPEG/PNG，优先 JPG/JPEG EXIF）
2. Free版最多 **20 张**，Pro版最多 **200 张**，超限需拦截并提示
3. 显示上传总数、已解析数、异常数
4. 缩略图网格展示
5. 支持清空全部 / 重传
6. 对 EXIF 缺失照片给出标识并启用回退策略

### EXIF 回退策略
- 优先：DateTimeOriginal
- 其次：CreateDate / ModifyDate
- 再次：文件修改时间（lastModified）
- 最后：标记为"EXIF缺失"

---

## 3.2 预览与水印定位模块
### 功能描述
用户可在预览区查看效果，并通过九宫格选择器或拖拽调整水印位置。

### 详细需求
1. 左侧预览区展示当前选中图片
2. 水印可拖拽（实时更新位置）
3. **可视化九宫格位置选择器**：
   ```
   ┌─────┬─────┬─────┐
   │ 左上 │ 上中 │ 右上 │
   ├─────┼─────┼─────┤
   │ 左中 │ 居中 │ 右中 │
   ├─────┼─────┼─────┤
   │ 左下 │ 下中 │ 右下 │ ← 默认
   └─────┴─────┴─────┘
   ```
4. 支持单位切换：**mm** 与 **px**
5. 默认边距：**2mm**
6. 横图、竖图统一使用相对位置策略

---

## 3.3 基础设置模块
### 3.3.1 时间格式（已全面升级）

支持以下格式组，每组包含完整变体（日期/日期时间/带秒/12小时制）：

| 分组 | 格式示例 |
|------|---------|
| **点号分隔（默认）** | `YYYY.MM.DD`, `YYYY.MM.DD HH:mm` ⭐默认 |
| **国际标准** | `YYYY-MM-DD (ISO)`, `YYYY-MM-DD HH:mm (ISO)` |
| **斜杠分隔** | `YYYY/MM/DD`, `YYYY/MM/DD HH:mm` |
| **美式格式** | `MM/DD/YYYY`, `MM/DD/YYYY hh:mm AM/PM` |
| **英式/欧式** | `DD/MM/YYYY`, `DD.MM.YYYY HH:mm` |
| **中文格式** | `YYYY年MM月DD日`, `YYYY年MM月DD日 HH:mm` |
| **紧凑格式** | `YYYYMMDD`, `YYYYMMDD HHmm` |
| **自定义** | 支持任意组合的占位符 |

#### 自定义格式占位符
- `YYYY` - 四位年份（2026）
- `MM` - 两位月份（03）
- `DD` - 两位日期（08）
- `HH` - 24小时制（18）
- `hh` - 12小时制（06）
- `mm` - 分钟（30）
- `ss` - 秒（45）
- `AM/PM` - 上下午标识

### 3.3.2 水印样式
- 字体：`"Courier New", "Lucida Console", monospace`
- 颜色：白色（`#ffffff`）
- 字号：16px
- 阴影：`0 1px 3px rgba(0,0,0,0.5)`

### 3.3.3 时间编辑能力
- 单张照片修改日期/时间
- 确认应用 / 恢复默认
- 修改后实时刷新预览

## 3.4 Pro 高级功能

### 3.4.1 时间批量修改 ⭐
**状态**：Pro 专属功能

- 批量加减天数
- 批量加减小时
- 应用到全部照片 / 批量恢复默认

### 3.4.2 宝宝成长水印 ⭐
**状态**：Pro 专属功能

#### 配置项
1. 出生日期（必填）
2. 宝宝昵称（可选）
3. **显示模式**：
   - 同时显示时间戳和成长印记（空格分隔）
   - 仅显示时间戳
   - 仅显示成长印记
4. **成长印记格式**：
   - X岁X个月
   - X个月X天
   - X岁X个月X天
   - 昵称 X岁X个月（带昵称）
   - 昵称 X个月X天（带昵称）
   - 昵称 X岁X个月X天（带昵称）

#### 年龄计算规则
- 以"拍摄日期 - 出生日期"计算
- 0~24个月：显示"X个月X天"
- >24个月：显示"X岁X个月"
- 若拍摄日期早于出生日期，提示：`照片时间早于宝宝出生，无法计算年龄`

#### 显示样式
成长印记与时间戳使用相同样式，合并为一行显示：
```
2026.03.08 18:30 宝宝 2岁3个月
```

## 3.5 批处理与导出模块（规划中）
### 功能描述
提供"分批处理"与"合并导出"能力。

### 详细需求

3. 导出方式：

   - 合并导出 ZIP（保留原文件名 + 后缀 `_timestamp`）

## 5. 业务规则与异常处理

| 场景 | 处理规则 |
|------|---------|
| 上传超过限制 | Free版>20张/Pro版>200张时拦截并提示升级 |
| 图片格式不支持 | 标记并跳过 |
| EXIF 缺失 | 使用文件修改时间，UI标识"EXIF缺失" |
| 时间格式非法 | 阻止保存并提示修正 |
| 用户未开通Pro点击高级功能 | 弹出订阅提示框 |

## 6. 非功能需求

### 6.1 性能
- 缩略图渲染 ≤ 3 秒
- 水印预览实时更新

### 6.2 兼容性
- 浏览器：Chrome / Edge / Safari 主流版本
- 终端：桌面端优先，兼容平板与手机

### 6.3 隐私与安全
- 前端本地处理，不上传服务器
- 使用浏览器 File API 读取 EXIF
