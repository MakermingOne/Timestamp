# 时光印记·Timestamp（照片批量时间戳添加工具）产品需求文档 PRD

> 版本: v1.9 | 最后更新: 2025-03-13

---

## 1. 产品概述

### 1.1 产品名称
**时光印记·Timestamp** - 照片批量时间戳添加工具

### 1.2 产品定位
时光印记是一款轻量化、无门槛的网页端工具，用于批量读取照片 EXIF 拍摄时间并自动添加时间戳/成长水印，覆盖个人照片整理、家庭相册冲印、生活记录归档等高频场景。

### 1.3 产品标语
> 时间水印批量添加，成长每一步都有迹可循

### 1.4 开发者信息
- **开发者**：小小铭 Makerming
- **版本**：v1.9
- **许可证**：MIT License

---

## 2. 版本范围与里程碑

### 2.1 Free 版（当前）
- 批量上传（最多 **20 张**）
- 自动读取 EXIF 拍摄时间
- 多种国际时间格式支持
- 可视化九宫格位置选择器
- 边缘距离自定义（默认 **3mm**）
- EXIF 信息展示
- 固定尺寸预览（600×400）
- 导出功能（当前/所有≤20张/打包 ZIP）
- 胶片风格时间戳字体

### 2.2 Pro 版（V1.9）
- 批量上传提升至 **200 张**
- 宝宝成长水印
- 宝宝头像 Logo（男宝/女宝/中性/不显示）
- 成长印记换行显示
- 单张/批量时间修改
- 时间批量偏移
- 批量压缩导出（可设置最大输出体积）
- 自定义时间格式

## 3.2 水印样式

### 3.2.1 时间戳字体
- **字体**：`"Courier New", "Courier", "Lucida Console", monospace`
- **字号**：14px
- **字重**：700
- **颜色**：白色（#ffffff）
- **阴影**：`0 1px 4px rgba(0,0,0,0.7), 0 2px 6px rgba(0,0,0,0.5)`
- **效果**：模拟旧式胶片相机时间戳风格

### 3.2.2 边缘距离
- **默认值**：3mm
- **单位选项**：mm / px
- **换算**：1mm ≈ 3.78px (at 96 DPI)

## 3.3 宝宝成长水印

### 3.3.1 头像 Logo 选择
| 选项 | 说明 | 样式 |
|------|------|------|
| 无 | 不显示头像（默认） | - |
| 默认 | 中性宝宝头像 | 白色矢量图 |
| 男宝 | 男宝宝头像 | 白色矢量图，小领结 |
| 女宝 | 女宝宝头像 | 白色矢量图，蝴蝶结 |

### 3.3.2 Logo 显示规则
- 尺寸：20x20px
- 位置：成长印记文字前
- 对齐：垂直居中
- 仅在选择了具体头像时显示

### 3.3.3 成长印记格式
- X岁X个月
- X个月X天
- X天
- X岁X个月X天
- 昵称 + 以上格式

## 3.4 批量压缩导出

### 3.4.1 功能说明
设置单张最大输出体积，超过限制的图片自动压缩。

### 3.4.2 可选体积
- 不限制（原图质量）
- 500 KB
- 1 MB
- 2 MB
- 5 MB（默认）
- 10 MB

### 3.4.3 Pro 限制
- Free 用户无法开启压缩导出
- Pro 用户可自由设置压缩参数

## 3.5 导出功能

### 3.5.1 导出按钮
| 按钮 | 功能 | Free 限制 |
|------|------|-----------|
| 导出当前 | 导出当前预览的照片 | 无限制 |
| 导出所有 | 导出所有照片 | ≤20张 |
| 打包导出 | 打包为 ZIP | ≤20张 |

### 3.5.2 导出限制逻辑
```javascript
function canExportAll() {
  if (photos.length <= 20) return true;
  return isPro;
}
```

## 3.6 预览功能

### 3.6.1 固定尺寸预览
- 预览画布：600px × 400px
- 使用缩略图显示（最大 600x400）
- 保持原始照片比例（contain 模式）

### 3.6.2 水印位置计算
- 基于真实照片尺寸计算
- 使用 contain 模式比例换算
- 预览位置与实际导出位置一致
