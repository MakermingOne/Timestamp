## 附录

### 测试数据准备

1. **带完整 EXIF 的图片**: 使用相机拍摄的原图
2. **无 EXIF 的图片**: 使用画图工具创建并保存
3. **大文件图片**: 高分辨率照片 (>10MB)
4. **不同格式**: JPG、JPEG、PNG

### 自动化测试建议

- 使用 Playwright 进行 E2E 测试
- 使用 Lighthouse 进行性能测试
- 使用 BrowserStack 进行跨浏览器测试
